# Barbearia API 

API RESTful desenvolvida em TypeScript para o gerenciamento de operações de uma barbearia, criada com foco em aprendizado e aplicação prática de backend.

 Tecnologias Utilizadas
- TypeScript
- Node.js
- MySQL(Banco de dados relacional)

📋 Funcionalidades
- Cadastro e gerenciamento de clientes.
- Controle de serviços oferecidos pela barbearia.
- Sistema de agendamentos integrados.
